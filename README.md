# 📚 Student Productivity Tracker

A simple Android application built using **Kotlin** and **XML** to help students manage daily tasks efficiently.

## ✨ Features
- 🔐 Login Screen
- 📊 Dashboard
- ➕ Add New Task
- 📋 View Tasks
- 🚪 Logout
- 🎨 Simple and Clean UI

## 🛠️ Technologies Used
- Kotlin
- XML
- Android Studio
- Gradle

## 🚀 Getting Started
1. Clone this repository.
2. Open it in Android Studio.
3. Sync Gradle.
4. Run the application on an emulator or Android device.
